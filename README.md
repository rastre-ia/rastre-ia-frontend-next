## Project setup

First install all the dependencies.

```bash
pnpm install
```

Now you need to create a `.env` file on root, this file contains the endpoints and API keys:

```
MONGO_DB_URL=
MONGO_DB_API_KEY=


```

## To run use:

```bash
pnpm dev
```

## To add shadcn componets:

```bash
npx shadcn@latest add <component-name>
```
